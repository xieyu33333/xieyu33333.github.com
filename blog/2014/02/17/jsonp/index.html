jQuery运用jsonp时的一个问题

Feb 17th, 2014

今天做一个小Demo的时候，需要跨域请求API，自然而然的想到jsonp,由于整个页面的数据来自于多个API接口，这时发现会报错：

TypeError: Property 'jsonpCallback' of object [object Object] is not a function

如果是异步加载的话，返回的值均为

jsonpCallback（{...json数据....}）

这样就造成了冲突，需要将与接口的异步通信改成同步通信，不过比较坑的是jQuery的Ajax函数内置的async参数对于jsonp是不生效的。
目前暂时先用回调嵌套的笨办法解决问题，先记录这个问题，研究出结果了继续写~
